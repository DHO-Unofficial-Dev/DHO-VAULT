# DHO Vault 0.2.0 릴리스 노트

DHO Vault `0.2.0`은 앱 내부 자동 업데이트, 작은 이미지의 서비스용 확대 저장, 전용 제목 표시줄과 상태 표시를 추가한 정식 버전입니다. 게임 파일은 이전과 같이 읽기만 하며, 분석과 이미지 변환은 사용자의 컴퓨터에서 처리합니다.

## 주요 변경 사항

### 앱 자동 업데이트

- 앱 시작 뒤 새 버전을 자동으로 확인합니다.
- 새 버전이 있으면 하단 상태 표시줄과 설정 화면에서 버전을 안내합니다.
- 설치판에서는 사용자가 선택한 뒤 업데이트를 내려받아 설치할 수 있습니다.
- 긴 릴리스 노트는 기본적으로 접어 두고 필요한 경우 제한된 영역 안에서 확인할 수 있습니다.
- 서버에서 받은 릴리스 노트는 HTML로 실행하지 않고 일반 텍스트로 표시합니다.
- 네트워크 연결이나 GitHub 접근에 실패해도 기존 이미지 탐색과 저장은 계속 사용할 수 있습니다.

`0.1.0`에는 자동 업데이트 기능이 없으므로 `0.2.0`은 GitHub Releases에서 직접 내려받아 설치해야 합니다. 이후 버전부터는 설치판에서 앱 내부 업데이트를 사용할 수 있습니다.

### 작은 이미지 저장 품질

- 48×48처럼 작은 정사각형 이미지는 `서비스용 선명 확대` 방식으로 저장하는 것이 기본값입니다.
- 가로·세로가 모두 64px 이하인 정사각형 이미지를 최소 192px까지 정수 배율로 확대합니다.
- Lanczos3 확대와 약한 선명화를 적용하며 투명 배경을 유지합니다.
- 설정에서 `원본 픽셀 유지 (무가공)`를 선택할 수 있습니다.
- 개별·현재 페이지·선택 항목·카테고리 전체 저장에 같은 설정을 적용합니다.
- 원본 게임 파일, 미리보기 썸네일, 업데이트 비교 기준점은 변경하지 않습니다.

### 앱 창과 상태 표시

- Windows 기본 제목 표시줄 대신 DHO Vault 전용 제목 표시줄을 사용합니다.
- 라이브러리·업데이트·설정 이동과 현재 버전, 최소화·닫기 버튼을 상단에 배치했습니다.
- 게임 폴더 연결과 이미지 저장 진행 상태를 하단에서 확인할 수 있습니다.
- 본문과 스크롤 영역이 상단·하단 고정 영역을 침범하지 않도록 정리했습니다.

## 출시 전 확인

- 실제 설치 환경에서 앱 내부 업데이트로 다음 버전을 발견하고 설치하는 전체 흐름을 확인했습니다.
- 업데이트 서명 검증, 설치 프로그램 실행과 새 버전 재실행을 확인했습니다.
- 업데이트 뒤 기존 게임 폴더 연결과 PNG 저장 품질 설정이 유지되는 것을 확인했습니다.
- 48×48 이미지를 기본 설정으로 저장했을 때 192×192로 확대되는 것을 확인했습니다.

## 다운로드 파일

- `DHO-Vault_0.2.0_x64-setup.exe`: 자동 업데이트를 지원하는 Windows 설치 파일
- `DHO-Vault_0.2.0_x64-portable.exe`: 설치하지 않고 실행하는 휴대용 파일
- `DHO-Vault_0.2.0_x64-setup.exe.sig`: 앱 업데이트 무결성 검증용 서명
- `latest.json`: 앱 자동 업데이트 메타데이터
- `SHA256SUMS.txt`: 배포 파일의 SHA-256 체크섬

자동 업데이트를 사용하려면 설치 파일을 권장합니다. 휴대용 파일은 새 버전을 안내할 수 있지만 설치판과 같은 자동 설치 동작을 보장하지 않습니다.

## 권장 환경

- Windows 11 x64
- Microsoft Edge WebView2 Runtime이 설치된 환경
- Papaya Play 대항해시대 온라인 한국 클라이언트의 현재 리소스 구조

Python, Node.js, Rust 같은 개발 도구는 필요하지 않습니다. WebView2가 없거나 설치할 수 없는 환경에서는 정상 실행을 보장하지 않습니다.

## 알려진 제한

- 자동 업데이트는 Windows NSIS 설치판을 기준으로 지원하고 검증합니다.
- 앱 업데이트 서명은 업데이트 파일의 무결성 확인용이며 Windows 코드 서명은 아닙니다. SmartScreen에서 게시자를 확인할 수 없다는 경고가 표시될 수 있습니다.
- Windows 10, WebView2 미설치 환경, 오프라인 설치 환경은 아직 검증하지 않았습니다.
- 게임 내 한글 아이템 명칭 검색은 아직 지원하지 않습니다.
- 의미를 확정하지 못한 이미지는 미분류 또는 넓은 카테고리로 표시될 수 있습니다.
- 실제 다음 게임 업데이트에서 신규 자산이 발생한 전체 비교 흐름은 아직 현장 검증 전입니다.

## 안전 및 권리 안내

DHO Vault는 게임 파일을 수정하거나 교체하지 않습니다. 이 프로그램과 저장소에는 게임 원본 파일이나 추출 이미지가 포함되지 않습니다.

DHO Vault는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다.
