# DHO Vault 0.1.0 릴리스 노트

DHO Vault `0.1.0`은 대항해시대 온라인 클라이언트에 포함된 이미지 리소스를 카테고리별로 찾아보고 PNG로 저장할 수 있는 첫 공개 버전입니다. 게임 파일은 읽기만 하며, 분석과 이미지 변환은 사용자의 컴퓨터에서 처리합니다.

## 주요 기능

- IM·IS·KP·SA·SB·SC·SD·SE·SF·SG·SH·TM·SW·SX·SY·SZ 아카이브의 이미지 탐색
- 사람이 확인한 카테고리와 검증된 지도·아트 타일 조립 결과 표시
- 카테고리, 아카이브, 이미지 ID, 블록 번호 검색
- 페이지 이동과 마지막으로 보던 위치 복원
- 상세 보기와 개별 PNG 저장
- 선택 이미지, 현재 페이지, 카테고리 전체 이미지 일괄 저장
- 마지막으로 선택한 게임 폴더 자동 연결
- 게임 업데이트 전후의 신규·변경·삭제 이미지 개수 비교
- 신규 이미지 목록 확인과 검토 후 비교 기준점 갱신

## 다운로드 파일

- `DHO-Vault_0.1.0_x64-setup.exe`: Windows 설치 파일
- `DHO-Vault_0.1.0_x64-portable.exe`: 설치하지 않고 실행하는 휴대용 파일
- `SHA256SUMS.txt`: 두 실행 파일의 SHA-256 체크섬

설치 파일과 휴대용 파일 중 하나만 받으면 됩니다. 실행하기 전에 같은 릴리스에 첨부된 `SHA256SUMS.txt`로 파일의 체크섬을 확인하는 것을 권장합니다.

## 권장 환경

- Windows 11 x64
- Microsoft Edge WebView2 Runtime이 설치된 환경
- Papaya Play 대항해시대 온라인 한국 클라이언트의 현재 리소스 구조

Python, Node.js, Rust 같은 개발 도구는 필요하지 않습니다. 설치 프로그램은 필요한 경우 WebView2 설치를 시도하지만, WebView2가 없거나 설치할 수 없는 환경에서는 정상 실행을 보장하지 않습니다.

## 처음 사용하기

1. 설치 파일 또는 휴대용 파일을 내려받습니다.
2. DHO Vault를 실행합니다.
3. 처음 나타나는 화면에서 `0010` 폴더를 포함한 대항해시대 온라인 게임 폴더를 선택합니다.
4. 라이브러리에서 카테고리를 선택하거나 검색하여 이미지를 확인하고 저장합니다.
5. 게임 업데이트 전 현재 상태를 기준점으로 저장하면, 이후 업데이트에서 새로 추가된 이미지를 따로 확인할 수 있습니다.

## 알려진 제한

- 실행 파일에 코드 서명이 적용되지 않아 Windows SmartScreen에서 게시자를 확인할 수 없다는 경고가 표시될 수 있습니다.
- Windows 11 x64와 WebView2가 설치된 환경에서 검증했습니다. Windows 10, WebView2 미설치 환경, 오프라인 설치 환경은 아직 검증하지 않았습니다.
- Papaya Play 한국 클라이언트의 현재 리소스 구조를 기준으로 검증했습니다. 다른 지역이나 이전 버전의 클라이언트는 정상 동작을 보장하지 않습니다.
- 게임 내 한글 아이템 명칭 검색은 아직 지원하지 않습니다.
- 의미를 확정하지 못한 이미지는 미분류 또는 넓은 카테고리로 표시될 수 있습니다.
- 타일 이미지는 사람이 검증한 조립 결과만 표시하며, 아직 확인하지 못한 조립 후보는 제공하지 않습니다.
- 자산 추가·변경·삭제 비교와 기준점 저장은 자동 테스트 및 현재 클라이언트로 검증했지만, 실제 다음 게임 업데이트에서 신규 자산이 발생한 전체 흐름은 아직 현장 검증 전입니다.

## 안전 및 권리 안내

DHO Vault는 게임 파일을 수정하거나 교체하지 않습니다. 이 프로그램과 저장소에는 게임 원본 파일이나 추출 이미지가 포함되지 않습니다.

DHO Vault는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다.
