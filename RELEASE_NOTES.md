# DHO Vault 0.2.0-rc.2 릴리스 노트

DHO Vault `0.2.0-rc.2`는 설치된 RC1에서 앱 내부 자동 업데이트 전체 흐름을 검증하기 위한 두 번째 시험 버전입니다. RC1의 자동 업데이트, 작은 이미지 서비스용 확대, 커스텀 창 기능을 모두 포함합니다.

## 변경 사항

### 업데이트 릴리스 노트 표시 개선

- 새 버전 안내는 대상 버전과 설치 가능 여부만 간결하게 표시합니다.
- 전체 릴리스 노트는 `릴리스 노트 보기` 영역에 기본적으로 접어 둡니다.
- 긴 릴리스 노트는 제한된 높이 안에서 스크롤하여 설정 화면 전체를 밀어내지 않습니다.
- 업데이트가 없거나 다시 확인하거나 설치가 끝나면 이전 릴리스 노트를 제거합니다.
- 서버에서 받은 릴리스 노트는 HTML로 실행하지 않고 일반 텍스트로 표시합니다.

## RC1에서 자동 업데이트 확인

1. 설치판으로 설치한 `0.2.0-rc.1`을 실행합니다.
2. 하단의 `v0.2.0-rc.2 업데이트` 안내를 선택하거나 설정 화면으로 이동합니다.
3. 자동으로 나타나지 않으면 `다시 확인`을 선택합니다.
4. `업데이트 설치`를 선택하고 다운로드 진행률과 설치 프로그램이 나타나는지 확인합니다.
5. 설치를 마친 뒤 DHO Vault를 다시 실행합니다.
6. 상단 버전이 `v0.2.0-rc.2`인지 확인합니다.
7. 기존 게임 폴더 연결과 PNG 저장 품질 설정이 유지되는지 확인합니다.

이 과정을 완료하면 RC 채널의 버전 비교, 다운로드, updater 서명 검증, 설치와 재실행을 실제 환경에서 확인할 수 있습니다. 문제가 생기면 아래 설치 파일을 직접 내려받아 RC2를 설치할 수 있습니다.

## 다운로드 파일

- `DHO-Vault_0.2.0-rc.2_x64-setup.exe`: 자동 업데이트 검증을 지원하는 Windows 설치 파일
- `DHO-Vault_0.2.0-rc.2_x64-portable.exe`: 설치하지 않고 실행하는 휴대용 파일
- `DHO-Vault_0.2.0-rc.2_x64-setup.exe.sig`: 앱 업데이트 무결성 검증용 서명
- `latest.json`: 앱 자동 업데이트 메타데이터
- `SHA256SUMS.txt`: 배포 파일의 SHA-256 체크섬

자동 업데이트 검증에는 설치 파일 사용을 권장합니다. 휴대용 파일은 새 버전 안내가 나타날 수 있지만 설치판과 같은 자기 업데이트 동작을 보장하지 않습니다.

## 권장 환경

- Windows 11 x64
- Microsoft Edge WebView2 Runtime이 설치된 환경
- Papaya Play 대항해시대 온라인 한국 클라이언트의 현재 리소스 구조

Python, Node.js, Rust 같은 개발 도구는 필요하지 않습니다. WebView2가 없거나 설치할 수 없는 환경에서는 정상 실행을 보장하지 않습니다.

## 알려진 제한

- 이 버전은 정식판이 아닌 시험용 RC입니다.
- 자동 업데이트는 Windows NSIS 설치판을 기준으로 지원하고 검증합니다.
- 앱 업데이트 서명은 업데이트 파일의 무결성 확인용이며 Windows 코드 서명은 아닙니다. SmartScreen에서 게시자를 확인할 수 없다는 경고가 계속 표시될 수 있습니다.
- Windows 10, WebView2 미설치 환경, 오프라인 설치 환경은 아직 검증하지 않았습니다.
- 게임 내 한글 아이템 명칭 검색은 아직 지원하지 않습니다.
- 의미를 확정하지 못한 이미지는 미분류 또는 넓은 카테고리로 표시될 수 있습니다.
- 실제 다음 게임 업데이트에서 신규 자산이 발생한 전체 비교 흐름은 아직 현장 검증 전입니다.

## 안전 및 권리 안내

DHO Vault는 게임 파일을 수정하거나 교체하지 않습니다. 이 프로그램과 저장소에는 게임 원본 파일이나 추출 이미지가 포함되지 않습니다.

DHO Vault는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다.
