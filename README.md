# DHO-VAULT

DHO Vault는 **대항해시대 온라인이 설치된 컴퓨터에서 게임 이미지 리소스를 찾아보고 PNG로 저장하는 Windows 프로그램**입니다.

게임 파일 안에는 장비 아이콘, 아이템 이미지, 지도와 아트 등 다양한 이미지가 들어 있지만 일반적인 이미지 파일처럼 바로 열어볼 수 없습니다. DHO Vault는 이러한 리소스를 읽어 사람이 확인한 카테고리별로 보여주고, 필요한 이미지를 선택해 저장할 수 있게 합니다.

> 현재 공개 배포판을 준비하고 있습니다. GitHub Releases에 실행 파일이 등록되기 전에는 일반 사용자가 바로 사용할 수 없습니다.

## 현재 구현된 기능

- 설치된 게임 폴더에서 IM·SA·SB·SC·SD·SE·SF·SG·SW·SX·IS 이미지 리소스 확인
- 사람이 확인한 카테고리별 이미지 탐색
- 카테고리, 아카이브, 이미지 ID, 블록 번호 검색
- 이미지 상세 보기와 개별 PNG 저장
- 현재 페이지와 카테고리 전체 PNG 일괄 저장
- 사람이 확인한 지도·아트 타일 조립 결과 표시
- 마지막으로 확인한 게임 폴더 자동 열기

아직 아이템의 게임 내 명칭으로 검색하는 기능은 지원하지 않습니다. 분류나 이미지 조립이 확실하지 않은 결과는 임의로 완성된 것처럼 표시하지 않으며, 확인된 범위만 순차적으로 추가합니다.

위 내용은 현재 소스 코드에 구현된 기능이며 첫 공개 버전의 확정 범위가 아닙니다. 공개 전까지 기능과 화면, 배포 방식은 달라질 수 있습니다.

## 게임 파일은 안전한가요?

DHO Vault는 게임 파일을 **읽기만 하며 수정하거나 교체하지 않습니다.** 모든 분석과 이미지 변환은 사용자의 컴퓨터에서 처리합니다.

이 저장소에는 게임 원본 파일이나 게임에서 추출한 이미지가 포함되지 않습니다. 프로그램을 사용하려면 사용자의 컴퓨터에 게임 클라이언트가 정상적으로 설치되어 있어야 합니다.

## 배포 계획

개발 도구가 없는 Windows 사용자가 실행할 수 있는 배포판을 준비하고 있습니다. 설치 파일과 휴대용 실행 파일을 후보로 검증하고 있지만, 첫 공개 버전과 포함 기능, 배포 파일 종류, 시스템 요구 사항은 아직 확정하지 않았습니다.

배포 범위가 확정되면 [GitHub Releases](https://github.com/DHO-Unofficial-Dev/DHO-VAULT/releases)에 실행 파일과 사용 방법을 함께 안내할 예정입니다. Python, Node.js, Rust 같은 개발 도구를 사용자에게 설치하도록 요구하지 않는 것을 목표로 합니다.

## 라이선스

DHO-VAULT의 소스 코드는 [Mozilla Public License 2.0](LICENSE)에 따라 공개됩니다.

이 라이선스는 DHO-VAULT가 직접 작성한 소스 코드에 적용됩니다. 게임 파일, 게임 데이터, 추출한 리소스 및 각 권리자의 상표에 대한 권리를 부여하지 않습니다.

## 안내

DHO-VAULT는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다. 이 프로젝트는 게임 원본 데이터나 추출 리소스를 배포하기 위한 프로젝트가 아닙니다.
