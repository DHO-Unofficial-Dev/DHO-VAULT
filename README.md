# DHO-VAULT

DHO-VAULT는 **대항해시대 온라인이 설치된 컴퓨터에서 게임 이미지 리소스를 찾아보고 저장하는 Windows 프로그램**입니다.

게임 파일 안에는 장비 아이콘, 소모품 이미지, 지도와 아트 등 다양한 이미지가 들어 있지만 일반적인 이미지 파일처럼 바로 열어볼 수 없습니다. DHO-VAULT는 이러한 리소스를 읽어 사람이 보기 편한 형태로 정리하고, 필요한 이미지를 선택해 저장할 수 있도록 만드는 도구입니다.

> 현재 공개 배포판을 준비하고 있습니다. GitHub Releases에 실행 파일이 등록되기 전에는 일반 사용자가 바로 사용할 수 없습니다.

## 제공할 기능

- 설치된 게임 폴더에서 이미지 리소스 찾기
- 장비, 아이템 등 확인된 카테고리별로 이미지 보기
- 많은 이미지를 빠르게 탐색하고 검색하기
- 원하는 이미지 미리보기
- 선택한 이미지를 PNG로 저장하기
- 여러 타일로 나뉜 지도와 아트를 올바른 형태로 합쳐 보기

지원하는 리소스와 카테고리는 실제 결과를 확인하면서 순차적으로 늘릴 예정입니다. 분류나 이미지 조립이 확실하지 않은 경우에는 임의로 완성된 결과처럼 제공하지 않습니다.

## 게임 파일은 안전한가요?

DHO-VAULT는 게임 파일을 **읽기만 하며 수정하거나 교체하지 않습니다.** 모든 분석과 이미지 변환은 사용자의 컴퓨터에서 처리하는 것을 원칙으로 합니다.

이 저장소에는 게임 원본 파일이나 게임에서 추출한 이미지가 포함되지 않습니다. 프로그램을 사용하려면 사용자의 컴퓨터에 게임 클라이언트가 정상적으로 설치되어 있어야 합니다.

## 사용 방법

배포가 시작되면 GitHub Releases에서 Windows용 실행 파일을 내려받아 사용할 수 있습니다. 구체적인 설치 및 사용 방법은 첫 공개 버전과 함께 안내할 예정입니다.

이 라이선스는 DHO-VAULT가 직접 작성한 소스 코드에 적용됩니다. 게임 파일, 게임 데이터,
추출한 리소스 및 각 권리자의 상표에 대한 권리를 부여하지 않습니다.

## 안내

DHO-VAULT는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다. 이 프로젝트는 게임 원본 데이터나 추출 리소스를 배포하기 위한 프로젝트가 아닙니다.
