# DHO-VAULT

DHO Vault는 **대항해시대 온라인이 설치된 컴퓨터에서 게임 이미지 리소스를 찾아보고 PNG로 저장하는 Windows 프로그램**입니다.

게임 파일 안에는 장비 아이콘, 아이템 이미지, 지도와 아트 등 다양한 이미지가 들어 있지만 일반적인 이미지 파일처럼 바로 열어볼 수 없습니다. DHO Vault는 이러한 리소스를 읽어 사람이 확인한 카테고리별로 보여주고, 필요한 이미지를 선택해 저장할 수 있게 합니다.

배포된 실행 파일은 [GitHub Releases](https://github.com/DHO-Unofficial-Dev/DHO-VAULT/releases)에서 받을 수 있습니다. 소스 코드를 직접 빌드하거나 개발 도구를 설치할 필요가 없습니다.

## 현재 구현된 기능

- 설치된 게임 폴더에서 IM·KP·SA·SB·SC·SD·SE·SF·SG·SH·TM·SW·SX·SY·SZ·IS 이미지 리소스 확인
- 사람이 확인한 카테고리별 이미지 탐색
- 카테고리, 아카이브, 이미지 ID, 블록 번호 검색
- 이미지 상세 보기와 개별 PNG 저장
- 페이지 이동과 마지막으로 보던 위치 복원
- 페이지와 카테고리를 넘나드는 이미지 선택 및 일괄 저장
- 현재 페이지와 카테고리 전체 PNG 일괄 저장
- 사람이 확인한 지도·아트 타일 조립 결과 표시
- 마지막으로 확인한 게임 폴더 자동 열기
- 게임 업데이트 전후의 신규·변경·삭제 이미지 개수 확인
- 신규 이미지만 모아보고 검토 후 비교 기준점 갱신

아직 아이템의 게임 내 명칭으로 검색하는 기능은 지원하지 않습니다. 분류나 이미지 조립이 확실하지 않은 결과는 임의로 완성된 것처럼 표시하지 않으며, 확인된 범위만 순차적으로 추가합니다.

위 내용은 첫 공개 버전 `0.1.0`을 기준으로 합니다. 이후 게임 클라이언트 변경과 사용자 검토 결과에 따라 지원 범위와 분류는 계속 보완될 수 있습니다.

## 게임 파일은 안전한가요?

DHO Vault는 게임 파일을 **읽기만 하며 수정하거나 교체하지 않습니다.** 모든 분석과 이미지 변환은 사용자의 컴퓨터에서 처리합니다.

이 저장소에는 게임 원본 파일이나 게임에서 추출한 이미지가 포함되지 않습니다. 프로그램을 사용하려면 사용자의 컴퓨터에 게임 클라이언트가 정상적으로 설치되어 있어야 합니다.

## 권장 실행 환경

DHO Vault는 **Windows 11 x64와 Microsoft Edge WebView2 Runtime이 설치된 환경**을 권장합니다. Python, Node.js, Rust 같은 개발 도구는 필요하지 않습니다.

설치 프로그램은 필요한 경우 WebView2 설치를 시도할 수 있지만, WebView2가 없거나 설치할 수 없는 환경에서는 DHO Vault의 정상 실행을 보장하지 않습니다.

## 다운로드 및 실행

GitHub Releases에는 다음 파일이 제공됩니다.

- `DHO-Vault_0.1.0_x64-setup.exe`: Windows 설치 파일
- `DHO-Vault_0.1.0_x64-portable.exe`: 설치하지 않고 실행하는 휴대용 파일
- `SHA256SUMS.txt`: 내려받은 실행 파일의 무결성을 확인하는 체크섬

설치 파일 또는 휴대용 파일 중 하나만 받으면 됩니다. 처음 실행한 뒤 대항해시대 온라인이 설치된 게임 폴더를 선택하면 리소스를 탐색할 수 있습니다.

### Windows 보안 경고

`0.1.0` 실행 파일은 코드 서명이 적용되지 않았습니다. 이 때문에 Windows SmartScreen에서 게시자를 확인할 수 없다는 경고가 표시될 수 있습니다.

실행하기 전에 반드시 이 저장소의 [GitHub Releases](https://github.com/DHO-Unofficial-Dev/DHO-VAULT/releases)에서 받은 파일인지 확인하고, 함께 제공되는 `SHA256SUMS.txt`와 체크섬을 비교하세요. 출처나 체크섬이 일치하지 않는 파일은 실행하지 마세요.

버전별 변경 사항과 알려진 제한은 [릴리스 노트](RELEASE_NOTES.md)에서 확인할 수 있습니다.

## 안내

DHO-VAULT는 비공식 팬 프로젝트입니다. 게임명, 상표 및 게임 데이터의 권리는 각 권리자에게 있습니다. 이 프로젝트는 게임 원본 데이터나 추출 리소스를 배포하기 위한 프로젝트가 아닙니다.
